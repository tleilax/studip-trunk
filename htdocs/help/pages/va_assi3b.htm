Termine

Haben Sie den zweiten Schritt zum Anlegen einer neuen, unregelmäßigen Veranstaltung erfolgreich abgeschlossen, sehen Sie in etwa folgendes Bild:





Dieses Beispiel zeigt den Typ der "unregelmässigen Veranstaltung".

Hier können Sie nun angeben:

  1. Daten über Termine
    Hier geben Sie bitte an, wann genau (Tag, Uhrzeit von-bis) die Veranstaltung stattfindet. Sie sehen zwei Felder für Veranstaltungstermine. Findet die Veranstaltung nur an einem Tag statt, lassen Sie das zweite Feld einfach leer. Gibt es mehr als zwei Sitzungen, klicken Sie auf die Schaltfläche Feld hinzufügen um Eingabefelder für den dritten oder vierten Termin anzulegen. Haben Sie aus Versehen einen Termin zuviel hinzugefügt, macht das nichts: neben einem hinzugefügten Termin erscheint eine Schaltfläche Löschen, mit dem Sie den überzähligen Termin wieder entfernen können.


  2. Keine Zeiten speichern
    Klicken Sie in das Kästchen, wenn Sie (noch) keine Veranstaltungszeiten angeben können oder wollen oder die Veranstaltung z.B. nach Absprache stattfinden soll. Sie können dann später die Veranstaltungszeiten z.B. über Termine bekanntgeben.


  3. Semester
    Gibt an, in welchem Semester die Veranstaltung beginnt/stattfindet.


  4. Endsemester
    ist nur bedeutsam für Veranstaltungen, die sich über mehrere Semester hinziehen; in der Regel also für Veranstaltungen des Typs "Organisation". Sie können Veranstaltungen bis unbegrenzt laufen lassen.


  5. Vorbesprechung
    Falls es einen Termin für eine Vorbesprechung gibt, tragen Sie ihn hier ein. andernfalls lassen Sie das Feld wie es ist.


Sind alle Eingaben korrekt, drücken Sie auf Weiter.
