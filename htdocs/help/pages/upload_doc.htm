Warum kann ich kein Word-Dokument (*.doc) hochladen?

Wir haben in Stud.IP ganz bewusst darauf verzeichtet, den Upload von Word-Dokumenten zu erlauben. Der Grund dafür ist die Dateikompatibilität: Word-Dokumente können leider nur von Windows und Macintosh Benutzern gelesen werden. Die Benutzer von anderen Betriebssystemen, dabei allen voran Linux, können mit diesem Format leider nichts anfangen. Daher sind in Stud.IP als Alternativen die Plattformunabhängigen Formate PDF (Acrobat Portable Document Format, *.pdf) und das Rich Text Format (*.rtf) erlaubt.

Wie kann ich stattdessen eine RTF-Datei erzeugen?

So funktioniert das Speichern einer RTF Datei unter Microsoft Word:

  • Wählen Sie das Menu Datei in der Kopfzeile
  • Wählen Sie den Eintrag Speichern unter aus
  • Klicken Sie in das Feld Dateityp und wählen Sie den Typ "Rich Text Format (*.rtf)" aus, wie im Bild gezeigt:



  • Klicken Sie auf Speichern

In anderen Word-Versionen, Textverarbeitungsprogrammen oder unter anderen Betriebssystem funktioniert das Erstellen von RTF Dateien ähnlich. Es können sich jedoch Abweichungen zum gezeigten Bild ergeben!
