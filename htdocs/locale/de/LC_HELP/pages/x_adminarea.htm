Ihre Veranstaltungen in Stud.IP zu verwalten ist sehr einfach. Auf den folgenden Seiten erfahren Sie alles über

  • die Änderung von Grunddaten


  • Änderungen von Zeiten


  • nachträgliches Erstellen oder Ändern des Ablaufplanes


  • Erstellen von Link- und Literaturlisten


  • Einstellen von News auf Ihrer Homepage oder der Veranstaltungsseite


  • Das Anlegen einer neuen Veranstaltung


Um Ihre Veranstaltungen zu verwalten gibt es eine "Verwaltungsecke" in Stud.IP: die Administrierungsseiten. Das hört sich sehr technisch an, aber keine Angst -es ist wirklich sehr einfach, sich als "Administrator" einer Veranstaltung zu betätigen.
Um auf die Administrierungsseiten zu gelangen, haben Sie mehrere Möglichkeiten.

Zugang 1

Klicken Sie auf Ihrer persönlichen Startseite, die Sie sehen wenn Sie sich in das System eingeloggt haben, auf Veranstaltungs-Verwaltung:



Zugang 2

Wählen Sie in der Kopfzeile

das Schlüsselsymbol an:

Der Schlüssel steht im System für "Administration". Mit einem Klick darauf erreichen Sie Ihre Administrierungsseite.



Zugang 3

Es gibt auch noch eine dritte Möglichkeit, Ihre Veranstaltungen zu verändern. Wenn Sie sich in der Veranstaltung befinden (über den Punkt Meine Veranstaltungen auf Ihrer persönlichen Startseite), finden Sie auf den Reiterkarten immer wieder Unterpunkte zur Administration, die nur Sie und vielleicht noch Ihre TutorInnen sehen. Befinden Sie sich z.B. auf der Übersichtsseite Ihrer Veranstaltung, sehen Sie einen Unterpunkt Administration dieser Veranstaltung. Klicken Sie darauf, landen Sie ebenfalls auf der Administrationsseite der Veranstaltung.



Auf der Administrierungsseite sehen Sie zunächst eine Reiterkarte mit dem Titel Dozenten und verschiedenen Unterpunkten.


Darunter finden Sie eine Liste aller Veranstaltungen, die Sie verwalten können.:


Sie können nun eine Veranstaltung mit der Schaltfläche

direkt anspringen. Sie befinden sich dann innerhalb der Veranstaltung und können dort mit den Unterpunkten auf der Reiterkarte die Bereiche Veranstaltungsdaten, Zeiten, Ablaufplan, Literatur/Links und News der ausgewählten Veranstaltung anwählen.



Sie können aber auch gleich auf der Administrierungsseite einen der genannten Unterpunkte anwählen. Klicken Sie z.B. den Unterpunkt Zeiten an, bekommen Sie eine Auswahlliste angezeigt, auf der Sie dann die Veranstaltung auswählen können, deren Zeiten Sie ändern wollen.


  Welchen Weg Sie wählen, ist hier letztlich egal. Machen Sie einfach, was Ihnen intuitiver erscheint.
